# Student Record Management System

This is a simple C++ project I made for managing student records.  
The program allows users to add, search, update, and view student information using a menu-driven system.

All student data is saved into a text file so the records can still be available after closing the program.

## Features

- Add new student records
- View all students
- Search student by ID
- Update student information
- Save records into a file
- Load saved records automatically

## Technologies Used

- C++
- File Handling
- Structures
- Functions
- Arrays

## File Used

- web.cpp → main source code
- students.txt → stores saved student records
